# VERIFY JVM COMPATIBILITY

Current crash:

```
VerifyError
Bad type on operand stack
```

This is NOT a normal exception.

The JVM rejected the generated bytecode before execution.

Meaning:

The generated class is invalid.

Possible causes:

Wrong superclass

Wrong field type

Wrong stack frame

Wrong stack map

Wrong invoke instruction

Wrong descriptor

Wrong lambda conversion

Wrong singleton generation

Wrong synthetic class generation
